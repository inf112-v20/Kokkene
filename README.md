## Requirements

An IDE running on either Windows 10, Mac osx or linux.

Java version 10+

## Documentation

1. Run Main
2. You will then be prompted by a menu.
3. Choose the settings you want.
4. Start game

## Shortcuts
    
            Exit game:      Q/Escape
    Toggle mute sound:      M
